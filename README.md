Este sitio web funciona como una solucion al Reto Factus de Halltec.

### ✨ Características principales:
Está app web esta optimizado para ofrecer una muestra del rapido proceso de la conexion entre Frontend, Backend y API.

- **Diseño:** Adaptado para una interfaz amigable con el usuario.
- **Sección de Formulario:** Datos directos al json que se envia a Strapi.

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este proyecto se utilizaron las siguientes herramientas:

* **REACT:** Estructura semántica del sitio.
* **CSS:** Estilos personalizados.
* **JavaScript:** Interactividad y funcionalidades dinámicas.
* **TypeScript:** Interactividad y funcionalidades dinámicas.
* **Strapi:** Procesamiento de datos y conexion con la Api.
